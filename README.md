# PG Life

A full-stack PG accommodation booking web application built using PHP, MySQL, HTML, CSS, JavaScript, Bootstrap, and React.

## Live Demo

🌐 https://pglife-rishabh.freepage.cc

## Features

- User Registration & Login
- Search PGs by City
- View Property Details
- Responsive Design
- React-based Property Listing
- MySQL Database Integration
- User Dashboard
- Mobile-Friendly Interface

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript, Bootstrap
- Backend: PHP
- Database: MySQL
- Additional: React
